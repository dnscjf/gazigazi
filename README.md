# 프로젝트 시작

## 기능 분배

- 윤성환 (내모임, 커뮤니티)
  - git 확인
- 박성호 (홈페이지, 모임개설, 모임검색)
- 황운철 (회원가입, 로그인, 마이페이지, 찜목록)
  - npm i -> start 정상작동 확인

## 라이브러리 설치

- axios
- swiper
- react-router-dom
- emotion
- react-icon
- react-calendar
